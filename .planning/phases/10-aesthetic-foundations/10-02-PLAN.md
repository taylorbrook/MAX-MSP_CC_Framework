    **Step 1: Add auto_size_panel() and is_complex_patch() to aesthetics.py.**

    `auto_size_panel(boxes: list[Box], padding: float = 18.0) -> tuple[float, float, float, float]`:
    - If boxes is empty, return (0.0, 0.0, 0.0, 0.0)
    - Compute bounding box: min_x = min(b.patching_rect[0] for b in boxes), min_y = min(b.patching_rect[1] for b in boxes), max_x = max(b.patching_rect[0] + b.patching_rect[2] for b in boxes), max_y = max(b.patching_rect[1] + b.patching_rect[3] for b in boxes)
    - Return (min_x - padding, min_y - padding, (max_x - min_x) + 2*padding, (max_y - min_y) + 2*padding)
    - Import Box using TYPE_CHECKING to avoid circular imports

    `is_complex_patch(patcher) -> bool`:
    - Complex if: len(patcher.boxes) >= 10, or any box has _inner_patcher not None (subpatcher present)
    - Uses object count + subpatcher presence as heuristic (Claude's discretion per CONTEXT.md)

    **Step 2: Add add_panel() method to Patcher class in patcher.py.**

    `add_panel(self, x: float, y: float, width: float, height: float, gradient: bool = True) -> Box`:
    - Use Box.__new__(Box) to create panel (not in DB, UI object)
    - Set ALL 16 fields: name="panel", args=[], id=self._gen_id(), maxclass="panel", text="", numinlets=1, numoutlets=0, outlettype=[], patching_rect=[x, y, width, height], fontname=FONT_NAME, fontsize=FONT_SIZE, presentation=False, presentation_rect=None, _inner_patcher=None, _saved_object_attributes=None, _bpatcher_attrs=None
    - Set extra_attrs: background=1, ignoreclick=1, border=0, rounded=7, mode=0
    - If gradient=True: add extra_attrs["bgfillcolor"] = {"type": "gradient", "color1": list(AESTHETIC_PALETTE["panel_fill"]), "color2": list(AESTHETIC_PALETTE["panel_gradient_end"]), "color": list(AESTHETIC_PALETTE["panel_fill"]), "angle": 270.0, "proportion": 0.39, "autogradient": 0}
    - If gradient=False: add extra_attrs["bgcolor"] = list(AESTHETIC_PALETTE["panel_fill"])
    - INSERT at index 0: self.boxes.insert(0, panel) -- NOT append. This is critical for z-order.
    - Return the panel Box

    - INSERT at index 0: self.boxes.insert(0, marker) -- background layer
