    - Return box

    - Set box.extra_attrs["fontface"] = FONTFACE_BOLD (1)
    - Set box.extra_attrs["textcolor"] = list(AESTHETIC_PALETTE["subsection_color"])
    - NO bgcolor (subsections don't have background color per CONTEXT.md)
    - Return box

    `add_annotation(self, text: str, x: float = 0.0, y: float = 0.0) -> Box`:
    - Call self.add_comment(text, x, y) to get base comment box
    - Set box.fontsize = 10.0
    - Set box.extra_attrs["fontface"] = FONTFACE_ITALIC (2)

    - Recalculate box.patching_rect[2] = len(text) * 6.0 + 14.0 (narrower chars at 10pt)
    - Set box.patching_rect[3] = 18.0 (shorter for 10pt)
    - Return box

    - Call self.add_comment(text, x, y) to get base comment box
    - Set box.extra_attrs["bubble"] = 1
    - Set box.extra_attrs["bubbleside"] = bubbleside if bubbleside is not None else BUBBLE_TOP
    - Adjust sizing for bubble chrome: box.patching_rect[2] += 17.0 (extra width for bubble), box.patching_rect[3] = 25.0 (bubble height)
    - Return box
