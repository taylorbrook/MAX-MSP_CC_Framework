  1. Create `src/maxpat/audit/cli.py`:
     - `DEFAULT_HELP_DIR = Path("/Applications/Max.app/Contents/Resources/C74/help")`
     - `DEFAULT_OUTPUT_DIR = Path(".claude/max-objects/audit")`

     - `main(argv: list[str] | None = None) -> int`:
       (a) Build argparse parser with:
           - `--help-dir` (Path, default DEFAULT_HELP_DIR): path to MAX help directory
           - `--output-dir` (Path, default DEFAULT_OUTPUT_DIR): output directory for audit results
           - `--outlets-only` (store_true): filter report to outlet type findings only
           - `--widths-only` (store_true): filter to width findings only
           - `--empty-io-only` (store_true): filter to empty I/O coverage only
           - `--connections-only` (store_true): filter to connection findings only
           - `--args-only` (store_true): filter to argument findings only
           - `--dry-run` (store_true): run analysis but don't write output files
           - `--verbose` (store_true): print progress during parsing
       (b) Parse args (from argv parameter, allowing test injection)
       (c) Validate help_dir exists, print error and return 1 if not
       (d) Initialize ObjectDatabase, HelpPatchParser, AuditAnalyzer, AuditReporter, OverrideGenerator
       (e) **Parse phase:** Call parser.parse_directory(help_dir). Collect all instances into flat list (all_instances) and group by object name (instances_by_object). Track parse_stats: files_parsed, files_failed (with error messages), total_instances, unique_objects.
       (f) **Filter phase:** Apply degenerate filtering using parser.filter_degenerate on each object's instances.
       (g) **Analyze phase:** Call analyzer.analyze_all(instances_by_object, all_instances). Get findings dict.
       (h) **Report phase:** Call reporter.generate_report(findings, parse_stats). If any --*-only flag set, call reporter.filter_report with the dimension. Call reporter.find_empty_io_objects(findings) for coverage data.
       (i) **Override phase:** Call override_generator.generate_proposed_overrides(findings, empty_io_data).
       (j) **Write phase (unless --dry-run):** Write audit-report.json, empty-io-coverage.json, proposed-overrides.json to output_dir.
       (k) Print summary to stdout: files parsed, objects analyzed, discrepancies found by confidence level, conflicts with manual overrides.
       (l) Return 0 on success.

     - Add `if __name__ == "__main__": sys.exit(main())` block.

  2. Update `src/maxpat/audit/__init__.py` to export cli.main for convenience:
     - Add `from .cli import main as run_audit` (guarded with try/except for circular import safety)

     - Test: `main(["--help-dir", "nonexistent/path"])` returns 1 (invalid path)

     - Test: `main(["--help-dir", str(fixtures_dir), "--output-dir", str(tmp_path), "--dry-run"])` runs the full pipeline on the fixture without errors
     - Test: Verify --outlets-only flag filters report to only outlet type findings
     - Use pytest tmp_path fixture for output directory isolation
