    - Import from maxclass_map (resolve_maxclass, is_ui_object), sizing (calculate_box_size), defaults (DEFAULT_PATCHER_PROPS, FONT_NAME, FONT_SIZE), db_lookup (ObjectDatabase).
    - Use dataclasses or simple classes (Claude's discretion on implementation detail).

    `class Box`:
    - Fields: id (str, auto-generated as "obj-N"), name (str, the MAX object name e.g. "cycle~"), args (list[str]), maxclass (str, resolved), text (str, name + args joined), numinlets (int), numoutlets (int), outlettype (list[str]), patching_rect (list[float] -- [x, y, w, h]), fontname (str), fontsize (float), presentation (bool, default False), presentation_rect (list[float] | None), extra_attrs (dict for any additional attributes).
    - Constructor takes: name, args=[], db=None (ObjectDatabase instance). If db provided, looks up object to get inlet/outlet counts and outlet types. If not found and name not in UI_MAXCLASSES, raises ValueError (Rule #1: never guess objects). Resolves maxclass via resolve_maxclass(). Computes size via calculate_box_size().
    - `to_dict() -> dict`: Returns the {"box": {...}} structure. For non-UI (maxclass=="newobj"): includes text, fontname, fontsize. For UI: includes parameter_enable: 0 (where applicable). Always includes: maxclass, id, numinlets, numoutlets, outlettype, patching_rect. If presentation is True, includes "presentation": 1 and "presentation_rect".
    - Special constructors or factory methods for comment, message, inlet, outlet (these are UI objects with specific behavior).

    `class Patchline`:
    - Fields: source_id (str), source_outlet (int), dest_id (str), dest_inlet (int), order (int, default 0), hidden (bool, default False).
    - `to_dict() -> dict`: Returns {"patchline": {"source": [source_id, source_outlet], "destination": [dest_id, dest_inlet], "order": order}}. If hidden, adds "hidden": 1.

    `class Patcher`:
    - Fields: boxes (list[Box]), lines (list[Patchline]), props (dict, copied from DEFAULT_PATCHER_PROPS), _next_id (int, auto-incrementing for unique Box IDs), db (ObjectDatabase instance -- created once, shared).
    - `__init__(self, db=None, is_subpatcher=False)`: If db is None, creates ObjectDatabase(). If is_subpatcher, uses SUBPATCHER_RECT for rect. Copies DEFAULT_PATCHER_PROPS via copy.deepcopy.
    - `add_box(name, args=None, x=0.0, y=0.0) -> Box`: Creates Box with auto-incremented ID, adds to boxes list. Position defaults to (0,0) -- layout engine sets real positions later.
    - `add_comment(text, x=0.0, y=0.0) -> Box`: Creates comment Box (maxclass "comment").
    - `add_message(text, x=0.0, y=0.0) -> Box`: Creates message Box (maxclass "message").
    - `add_connection(src_box, src_outlet, dst_box, dst_inlet, order=0) -> Patchline`: Creates Patchline using box IDs.
    - `add_subpatcher(name, inlets=1, outlets=1) -> tuple[Box, Patcher]`: Creates a Box with maxclass "newobj", text "p {name}", and an embedded Patcher (is_subpatcher=True). The inner Patcher gets inlet/outlet objects added automatically. Returns (parent_box, inner_patcher). Parent box numinlets/numoutlets set to match inlet/outlet count.
    - `add_bpatcher(filename=None, embedded=False, args=None) -> Box | tuple[Box, Patcher]`: If filename provided, creates bpatcher box referencing external file. If embedded=True, creates bpatcher with inner Patcher and returns tuple. Uses bpatcher-specific attributes (bgmode, border, clickthrough, etc.) from research Pattern 5.
    - `to_dict() -> dict`: Returns {"patcher": {**self.props, "boxes": [box.to_dict() for box in self.boxes], "lines": [line.to_dict() for line in self.lines]}}. For subpatcher boxes, the embedded patcher's to_dict() result goes into the box's "patcher" field. Includes "saved_object_attributes" for subpatcher boxes.
