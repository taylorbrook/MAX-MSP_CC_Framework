  <name>Task 1: Create public API and file write hooks</name>
  <files>src/maxpat/__init__.py, src/maxpat/hooks.py, tests/test_hooks.py</files>
  <behavior>
    - generate_patch(patcher) applies layout, runs validation, returns (dict, list[ValidationResult])
    - generate_patch raises PatchGenerationError if unfixable structural errors exist
    - generate_patch with valid patcher returns (.maxpat dict, results with warnings/info only)
    - write_patch(patcher, path) calls generate_patch then writes JSON to path
    - write_patch returns list[ValidationResult] so caller sees warnings
    - write_patch creates parent directories if they don't exist
    - write_patch blocks (raises PatchValidationError) if has_blocking_errors is True
    - write_patch uses json.dumps with indent=2 for readable output
    - validate_file(path) loads .maxpat from disk, runs validate_patch on the dict, returns results
    - validate_file raises FileNotFoundError if path doesn't exist
    - from src.maxpat import Patcher, Box, Patchline, generate_patch, write_patch, validate_file -- all importable
  </behavior>
  <action>
    Create `src/maxpat/hooks.py`:
    - `class PatchGenerationError(Exception)`: raised when unfixable errors prevent generation.
    - `class PatchValidationError(Exception)`: raised when validation blocks file write.
    - `def write_patch(patcher, path, validate=True) -> list[ValidationResult]`:
      1. Call generate_patch(patcher) to get (patch_dict, results).
      2. If validate is True and has_blocking_errors(results): raise PatchValidationError with the error messages.
      3. Create parent directories with Path(path).parent.mkdir(parents=True, exist_ok=True).
      4. Write json.dumps(patch_dict, indent=2) to path.
      5. Return results (warnings/info for caller to inspect).
    - `def validate_file(path) -> list[ValidationResult]`:
      1. Load JSON from path. If invalid JSON, return [ValidationResult("json", "error", "Invalid JSON: {err}")].
      2. Run validate_patch on the loaded dict.
      3. Return results.

    Update `src/maxpat/__init__.py`:
    - `from .patcher import Patcher, Box, Patchline`
    - `from .validation import validate_patch, ValidationResult, has_blocking_errors`
    - `from .layout import apply_layout`
    - `from .hooks import write_patch, validate_file, PatchGenerationError, PatchValidationError`
    - `from .db_lookup import ObjectDatabase`
    - `def generate_patch(patcher) -> tuple[dict, list[ValidationResult]]`:
      1. apply_layout(patcher)
      2. patch_dict = patcher.to_dict()
      3. results = validate_patch(patch_dict, db=patcher.db)
      4. If has_blocking_errors(results): raise PatchGenerationError with error details.
      5. Return (patch_dict, results)
    - `__all__` listing all public names.

    Create `tests/test_hooks.py`:
    - write_patch creates a .maxpat file on disk (use tmp_path fixture)
    - Written file is valid JSON loadable by json.loads
    - write_patch creates parent directories
    - write_patch blocks on unfixable errors (test with corrupted patcher)
    - write_patch returns validation results (warnings visible)
    - validate_file loads and validates existing .maxpat file
    - validate_file raises FileNotFoundError for missing file
    - validate_file returns errors for invalid JSON file
    - write_patch with validate=False skips validation
  </action>
  <verify>

  </verify>
